Quick Actions

Auto-Maintain
Who wants to worry about culling lengthy lists of completed tasks or emptying the trash? No one. That's why the Tasks app does it for you. Set how many tasks you want to remain in your completed task list and trash. Tasks will take care purging aging tasks.
Global Shortcuts
esc Perform context aware action
⌘ N Add task
⌘ ⇧ S Open Snoozed
⌘ ⇧ T Open Trash
⌘ , Open Settings
NOTE: The esc key action is based on context. When viewing Snoozed, Trash, or Settings, esc will return the user to the Tasks page. When on the Tasks page, esc will close the sidebar (if visible and the Tasks Sidebar setting is set to "Open on double click") or set cursor focus to the "Add tasks" input box.
